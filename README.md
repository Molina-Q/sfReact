# Boilerplate code for project using Symfony UX React

## Requirement
- PHP
- Scoop
- Composer
- Symfony CLI
- Node.js

## 1. Install dependencies:
  ``` composer

  composer install
    
  ```
   
  ```npm

  npm install
   
  ```
## 2. launch the project

 ```composer
 
 symfony serv:start -d
 
 ```
 
 ```npm
 
 npm run dev
 
 ```
